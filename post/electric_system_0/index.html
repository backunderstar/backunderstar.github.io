『电力系统分析』绪论

发表于2023-04-06|更新于2023-04-07|学习与笔记电力系统分析
|字数总计:1.4k|阅读时长:4分钟|阅读量:

电力系统组成

基本结构

  • 电力系统=发电厂+电力网+电力用户
  • 实际应用中:“电力网”和“电力系统”有时不加区分,指相同的意思。

电力系统组成

  • 电力网:变压器和不同电压等级输电线路组成的网络
  • 电力系统:发电机、电力网及用电设备按一定规律连接而成的系统
  • 动力系统:电力系统+发电厂的动力部分(火力发电厂的 锅炉、汽轮机和水力发电厂的水库、水轮机以及核动力 发电厂的反应堆等)

电力系统基本环节

  • 发输变配用—发电、输电、变电、配电、用电
  • 发电、输电、变电、配电、用电的所有装置和设备仪器组成了电 力系统

电力系统运行的特点和要求

电力系统运行特点

  1. 电能不能大量存储:电能的生产、变换、输送、分配和使用必须同时进行
  2. 过渡过程十分短暂:要求自动化控制程度高
  3. 对社会生活、政治和国民经济影响巨大

电力系统基本要求

  • 保证安全可靠持续供电
  • 保证必要的电能质量
  • 保证电力系统自身运行的安全性
  • 保证电力系统运行的经济性

电能质量指标

频率

  • 额定频率:50Hz
  • 频率偏差和延时时间:+-0.2Hz,持续时间小于30分钟

供电电压

  • 电压偏差:
    • 36KV及以上,正负偏差的绝对值之和小于额定电压的10%
    • 20KV及以下,正负偏差的绝对值之和小于额定电压的7%
    • 220V额定电压的**+7%、-10%**
  • 三相不平衡度:小于2%

谐波

  • 正弦波电压和电流有谐波畸变率的限制

电力系统发展

早期电力系统

  • 1882年,爱迪生创立了Pearl St. Station
  • 1882年,德国建立第一条直流输电线路, 1500~2000V,输 电效率<25%
  • 1884年,Frank J Sprague 发明了直流电动机
  • 1885年,William Stanley发明了商用变压器
  • 1888年,俄国人多布洛奥斯基发明三相交流制和效率很高的 三相异步电动机
  • 1891年,德国建立了世界第一条三相交流高压输变电系统, 12kV、175km输电线路,输电效率达到80%

中国电力系统发展

  • 发电机装机容量:最新数据显示,截至2020年底,全国发电装机容量22.0亿千瓦, 同比增长9.5%,年均增长7.6%。(来源:中国电力知库 ID: pkthinker)
  • 输变电网络建设
  • 电力体制改革
    • 厂网分开:发电与电网公司
    • 主辅分离:能源与电能
  • 区域电网分布互联:西电东送

电力系统中的发电厂

发电厂

  • 生产电能的工厂,把天然能源转化电能

分类

  • 按发电厂能源取得的方式不同分类
    • 火力发电厂—70%
    • 水力发电厂—20%
    • 核电厂
    • 其他形式
      • 小容量:柴油机厂
      • 一次能源发电:太阳能、风力、潮汐、地热电厂

电压等级与额定电压

电压等级

  • 电压等级(线路及用电设备额定电压): 3kV,6kV, 10kV,35kV,66kV,110kV,220kV,330kV, 500kV, 750kV,1000kV
    • 输电网:220kV级以上主干电力线路
    • 二级输电网:110kV ~ 220kV,连接区域性的发电厂和大用户
    • 高压配电网:6kV ~ 35kV
    • 低压配电网:1kV及以下电网

设备额定电压设定的基本原则

  • 电力系统的额定电压:根据技术经济条件规定的电力系统电压等级标准,又称电力网或线路的额定电压。有3kV, 6kV, 10kV, 35kV,(60kV) 110kV, (154kV),220kV, 330kV, 500kV等电压等级

  • 用电设备额定电压与电力系统额定电压相同

  • 用电设备电压允许偏差:±5%

电力系统中性点运行方式

电力系统中性点

  • 电力系统中性点:电力系统三相交流发电机、变压器星形绕组的公共点
  • 中性点接地(运行)方式:中性点与大地间的电气连接方式

中性点接地方式类型

  • 有效接地系统或大接地电流系统

    • 中性点直接接地、经小阻抗接地
      • 直接接地
      • 经小电阻
      • 经小电抗
    • 过电压水平和输变电设备所需的绝缘水平较低,但是系统发生单相接地短路故障时短路电流很大, 用于110kV及以上的高压电力系统中。

  • 非有效接地系统或小接地电流系统

    • 中性点不接地、经消弧线圈接地、经高阻接地
      • 不接地
      • 经消弧线圈(电抗)
      • 经高阻
    • 发生单相接地短路故障时短路电流很小,故障相对地电压降为零;非故障相对地电压升高为线电压其值主要由电容电流组成, 用于110 kV以下的低压电力系统中。
