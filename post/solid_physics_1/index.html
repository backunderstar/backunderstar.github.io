『固体物理』1 晶体结构

发表于2023-05-07更新于2023-05-07
字数总计:2.5k阅读时长:9分钟阅读量: 成都

晶体的特征

固体的分类

按结构分类 特性
晶体 长程有序：单晶体、多晶体
非晶体 不具有长程有序的特点，短程有序
准晶体 有长程取向性，没有长程的平移对称性
  • 长程有序：至少在微米量级范围内原子排列具有周期性

晶体的宏观特性

  1. 自限性：自发形成封闭凸多面体

  2. 解理性：晶体沿某些确定方位的晶面劈裂的性质，这样晶面为解理面。

  3. 晶面角守恒定律：同一物质的不同晶体在同一温度和压强下，对应晶面之间的夹角恒定。

  4. 晶体的各向异性：不同方向，晶体物理性质不同，原子排列情况不同导致。

  5. 晶体的均匀性：同一方向上任意两点物理性质相同

  6. 晶体的对称性：几个特定方向异向同性，相同的性质在不同方向有规律出现。

  7. 晶体固定的熔点：某一温度开始融化，期间温度不变，全部融化，温度才开始上升。

  • 晶棱、晶带、带轴、晶轴的概念

  • 晶体的宏观特性是由晶体内部结构的周期性决定的,即晶体的宏观特性是微观特性的反映。

晶体的结构

晶体结构的周期性

概念：一个理想的晶体是由完全相同的结构单元在空间周期性重复排列而成的。

  • 基元：在晶体中适当选取某些原子作为一个基本结构单元，这个 基本结构单元称为基元。
  • 晶格：晶体的内部结构可以概括为是由一些相同的点子在空间有规则地做周期性无限分布，这些点在空间周期性排列形成的骨架称为晶格。
  • 格点：晶格中的点子代表着晶体结构中相同的位置，称为格点。一个格点代表一个基元。

晶格+基元=晶体结构

  • 布拉菲晶格：格点的总体称为布拉菲晶格，这种格子的特点是每点周围的情况完全相同。
  • 简单晶格：如果晶体由完全相同的一种原子组成，且每个原子周围的情况完全相同。
  • 复式晶格：两种或两种以上原子组成，同种原子构成的网格为子晶格，相对位移形成复式晶格。

原胞

原胞：晶格中取一个格点，以三个不共面方向周期为边长的平行六面体形成重复单元，这个平行六面体沿着三个不同方向周期性平移，可以充满晶格，这个平行六面体就是原胞。

基矢：原胞三个边的矢量，基本平移矢量。

配位数

  • 一个粒子周围最近邻的粒子数称为配位数
  • 可以描述晶体中粒子排列的紧密程度，粒子排列越紧密， 配位数越大

配位数的可能值：12(密堆积)，8(氯化铯型结构)，6(氯化钠型结构)，4(金刚石型结构)，3(石墨层状结构)，2(链状结构)。

密堆积

如果晶体由完全相同的一种粒子组成，而粒子被看作小圆球，则这些全同的小圆球最紧密的堆积称为密堆积

  1. 六角密积：
    1. 第一层：每个球与6个球相切，有6个空隙， 如编号1,2,3,4,5,6。
    2. 第二层：占据1,3,5空位中心。
    3. 第三层：与第一层位置对应。
    4. 六角密积是复式格，其布拉维晶格是简单六角晶格。
    5. 基元是两个原子：(000)(0 0 0)(000)和(231312)(\frac{2}{3} \frac{1}{3} \frac{1}{2} )(32​31​21​)，即r⃗=23a⃗+13b⃗+12c⃗\vec{r} =\frac{2}{3} \vec{a}+\frac{1}{3} \vec{b}+\frac{1}{2} \vec{c}r=32​a+31​b+21​c
  2. 立方密积：
    1. 第一层：每个球与6个球相切，有6个空隙，如编号为1,2,3,4,5,6。
    2. 第二层：占据1，3，5空位中心。
    3. 第三层：占据2，4，6空位中心。
    4. 第四层：与第一层位置对应。
    5. 形成面心立方结构，称为立方密积。
    6. 特点：结合能低，晶体结构稳定；配位数最大为12。

致密度

原子看做硬球，硬球尽可能大并相切，晶胞中，原子硬球占据的体积和晶胞体积之比为致密度。

小结

  • 每一个布拉菲格点几何上是等价的；站在任何一点上看周围的情况都相同；eg:面心立方上所有点都是等价的；几何特征都是面心立方(结构形式)，只是点与点距离是不一样。
  • ABO3ABO_{3}ABO3​型钙钛矿结构，基元包括5个不等价的原子，基元取ABO3ABO_{3}ABO3​。
  • 密堆积：原子在晶体中的平衡位置，相应于体系能量最低的 位置，因此总是尽可能地紧密排列。
  • 配位数：最近邻的原子数，不必是同种原子，但距离相同。
  • 原胞：最小的基元，一个格点，原胞只考虑平移的对称性。 晶胞：结晶学上用的基元，包含一个或几个原胞，保持晶体宏观对称性，同一晶体原胞的形状不是唯一的，晶胞是唯一的。
  • 反映几何特征的立方块：晶胞/单包/单晶包/布拉菲单包；由基矢决定的体积最小单元：原胞；范围小，理论计算方便，范围大，充分体现结构特征。
