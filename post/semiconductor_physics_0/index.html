『半导体物理』绪论

发表于2023-04-04|更新于2023-04-04|学习与笔记半导体物理
|字数总计:2k|阅读时长:6分钟

写在前面：

​ 绪论内容好像不是很重要，只是对这门课程基础的基础有个了解，宏观上有个把握感觉就差不多。

课程内容

理论

  • 半导体的晶体结构和电子状态
  • 杂质和缺陷能级
  • 载流子的统计分布
  • 载流子的散射及电导问题
  • 非平衡载流子的产生、复合及运动规律

应用、器件结构

  • 半导体的表面和界面 （pn结、金属与半导体接触、半导体表面与 MIS结构、半导体异质结构）
  • 半导体的光、热、磁、压阻等物理现象
  • 非晶半导体

什么是半导体

导电性

  • 电阻率ρ介于导体和绝缘体之间，并且具有负的电阻温度系数→半导体
    • 导体：
    • 半导体：
    • 绝缘体：

电阻温度系数

  • 半导体电阻值是负的温度系数

电子激发能隙

  • 导 体： Eg ~ 0 例如： Eg (Cu) ~ 0 eV
  • 绝缘体： Eg > 3 eV 例如： Eg (SiO2 )=8.0 eV
  • 半导体： 0 < Eg < 3 eV 例如： Eg (Si) =1.12 eV
  • 例外：
    • 半导体金刚石 ( Eg ~ 6 eV )
    • 氮化镓 GaN ( Eg ~ 3.5 eV )
    • 硫化锌 ZnS ( Eg ~ 3.5 eV )

半导体材料的分类

功能和应用 组成 结构
微电子半导体、光电半导体、
热电半导体、微波半导体、
气敏半导体		 无机半导体：元素、
化合物 Si、Ge、GaN、InP……
有机半导体		 晶体：单晶体、多晶体
非晶：无定形，短程有序、长程无序结构

无机半导体晶体材料

  • 元素半导体
  • 化合物半导体
  • 固溶体半导体：具有半导体性质的固溶体，又称混晶或混合晶体

  • 下面是一些具体的例子：
  • 过渡金属氧化物半导体：有ZnO、SnO2、V2O5、Cr2O3、Mn2O3、 FeO、CoO、NiO等。
  • 尖晶石型化合物（磁性半导体）：主要有CdCr2S4、CdCr2Se4、 HgCr2S4等。
  • 稀土氧、硫、硒、碲化合物：有EuO、EuS、EuSe、EuTe 等。

非晶态半导体

  • 都是非结晶体，它们缺乏构成原子 周期性的长程序
  1. 非晶Si、非晶Ge以及非晶Te、Se元素半导体
  2. 化合物有GeTe、As2 Te3、Se4 Te、Se2 As3、As2 SeTe非晶半导体
  3. 非晶态与晶态半导体具有类似的基本能带结构，也有导带、 价带和禁带

有机半导体

  • 具有半导体性质的有机材料
  1. 有机半导体可分为有机物、聚合物、给体-受体络合物三 类。
    1. 有机物类包括芳烃、染料、金属有机化合物，如紫精、酞 菁、孔雀石绿、若丹明B等。
    2. 聚合物类包括主链为饱和类聚合物和共轭型聚合物，如聚 苯、聚乙炔、聚乙烯咔唑、聚苯硫醚等。
    3. 电荷转移络合物由电子给予体与电子接受体二部分组成， 典型的有四甲基对苯二胺与四氰基醌二甲烷复合物。

几种主要半导体发展现状与趋势

硅

  • 直拉法（ CZ 法）
  • 区熔法（ FZ 法）
  • 外延法
  • 增大直拉硅（CZ-Si）单晶的直径仍是今后CZ-Si发展的总趋势

GaAs和InP单晶

以低位错密度生长的2～3英寸的导电GaAs衬底材 料为主。

  • InP比GaAs

    • 具有更优越的高频性能
    • 发展的速度更快

  • 但是

    • 研制直径3英寸以上大直径的InP 单晶的关键技术 尚未完全突破，价格居高不下。

半导体超晶格、量子阱

半导体超晶格是指由一组多层薄膜周期重复排列而成的单晶。

  • III-V族超晶格、量子阱材料 GaAlAs/GaAs，GaInAs/GaAs， AlGaInP/GaAs； GaInAs/InP， AlInAs/InP， InGaAsP/InP等
  • GaAs、InP基晶格匹配和应变 补偿材料体系已发展得相当成熟。
  • 已成功地用来制造超高速、 超高频微电子器件和单片集成电路。

硅基应变异质结构材料

  • 目前硅基材料研究的主流：GeSi/Si应变层超晶格材料 → 新一代移动通信。

  • Si/GeSi MOSFET 的最高截止频率已达200GHz，噪音在 10GHz下为0.9dB，其性能可与GaAs器件相媲美。

一维量子线、零维量子点

  1. 基于
    1. 量子尺寸效应
    2. 量子干涉效应
    3. 量子隧穿效应
    4. 非线性光学效应
  2. 应用
    1. 低维新型半导体材料
    2. 人工构造（通过能带工程实施）
    3. 新一代量子器件的基础

宽带隙半导体材料

金刚石 碳化硅 III族氮化物、立方氮化硼 II-VI族硫、锡、碲化物、氧化物及固溶体等

  • 高频大功率、耐高温、抗辐射半导体微电子 器件和电路的理想材料
  • 通信、汽车、航空、航天、石油开采及国防

半导体材料的地位

  • 半导体微电子和光电子材料是很多科学技术的基础
  • 比如通信、高速计算、 大容量信息处理、 空间防御、电子对 抗、武器装备的微 型化、智能化
  • 而这些科学技术都与国民经济与国家安全息息相关

半导体微电子和光电子材料 —> 科学技术 —> 国民经济、国家安全

半导体的发展

第一代半导体材料

  • 主要是指硅（Si）、锗元素（Ge）半导体材料

​ 作为第一代半导体材料的锗和硅，在国际信息产业技术 中的各类分立器件和应用极为普遍的集成电路、电子信 息网络工程、电脑、手机、电视、航空航天、各类军事 工程和迅速发展的新能源、硅光伏产业中都得到了极为 广泛的应用，硅芯片在人类社会的每一个角落无不闪烁 着它的光辉。

第二代半导体材料

  • 主要是指化合物半导体材料

​ 如砷化镓（GaAs）、锑化铟（InSb）；三元化合物半导 体，如GaAsAl、GaAsP；还有一些固溶体半导体，如GeSi、GaAs-GaP；玻璃半导体（又称非晶态半导体），如 非晶硅、玻璃态氧化物半导体；有机半导体，如酞菁、 酞菁铜、聚丙烯腈等。

第三代半导体材料

  • 主要以碳化硅（SiC）、氮化镓（GaN）、氧化锌 （ZnO）、金刚石、氮化铝（AlN）为代表的宽禁带半导体材料。

​ 在应用方面，根据第三代半导体的发展情况，其主要应 用为半导体照明、电力电子器件、激光器和探测器、以 及其他领域，每个领域产业成熟度各不相同。在前沿研 究领域，宽禁带半导体还处于实验室研发阶段。

第三代半导体特征及应用

  • 第一代 硅（Si）等奠定微电子产业基础
  • 第二代 砷化镓（GaAS）等奠定信息产业基础
  • 第三代半导体支撑绿色、低碳、智能社会的可持续发展

五高特性

第三代半导体材料（氮化镓、碳化硅、金刚石等）具有五高特性

  1. 高光效
  2. 高功率
  3. 高电压
  4. 高频率
  5. 高工作温度

氮化镓的唯一性

  • 目前已知的可达到万伏千安等级（全球能源互联网必须的超特高压柔性直流输电）的唯一功率半导体材料

  • 目前能实现高频、高效、大功率（5G通信要求频率覆盖到40GHz以上，输出功率几瓦到几百瓦）的唯一材料

  • 以氮化镓为代表的氮化物是唯一覆盖可见光到紫外波长范围的半导体发光材料体系
